# Como rodar

É recomendado ter um ambiente virtual Ruby configurado, porém, apenas as ferramentas base já bastam.

- Clonar repositório;
- Instalar gemas com `bundle install`
- Rodar migrações com `rake db:migrate`
- Rodar servidor com `rails server`
